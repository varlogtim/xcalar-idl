Excel-equivalent functions

This section lists the Xcalar functions that accomplish the same tasks as some of the most commonly used Excel functions. If an equivalent Xcalar function is not available, this section suggests an alternate method. Also, if the Xcalar function works differently than the corresponding Excel function, a brief description of the Xcalar function is provided.

Excel function category: Compatibility

Excel Function Excel Description Xcalar Map function Alternate method Xcalar syntax
FLOOR Rounds a number down, toward zero. floor  

map(floor(operand))

Returns the largest integer less than or equal to the input. Equivalent to the Excel function when the multiple is 1.
STDEV Estimates standard deviation based on a sample.   Profile  

Excel function category: Date and time

Excel function Excel description Xcalar Map function Xcalar syntax
NOW Returns the serial number of the current date and time. default:now

map(default:now())

Returns the current time of the XDP cluster.

Excel function category: Logical

Excel function Excel description Xcalar Map function Xcalar syntax
AND Returns TRUE if all of its arguments are TRUE. and map(and(operand1,operand2))
IF Specifies a logical test to perform. if map(if(condition, return value, return value if false))
NOT Reverses the logic of its argument. not map(not(operand))
OR Returns TRUE if any argument is TRUE. or map(or(operand1,operand2))

Excel function category: Lookup and reference

Excel function Excel description Alternate method
VLOOKUP Looks in the first column of an array and moves across the row to return the value of a cell.

Join operation to join two tables using matching column(s).
