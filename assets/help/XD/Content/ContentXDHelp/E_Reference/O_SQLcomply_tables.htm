Xcalar SQL compliance

This topic provides the list of the features and subfeatures as described in the SQL ANSI standard and the type of support level Xcalar provides for each feature.

NOTE: This list is for information purposes only. If a subfeature is not fully supported then the main feature is listed as Partial support.

The following list describes the support levels provided by Xcalar:

  • Full support: Supports the entire feature.
  • Partial support: Supports some but not all of the standard subfeatures.
  • Enhanced support: Supports the standard semantics, including additional functionality.
  • Equivalent support: Supports the standard semantics with non-standard syntax.
  • Similar support: Supports a similar function with non-standard semantics and syntax.
  • Not supported: Does not support this feature.
